## Технологии
- [Python v.3.10]()
- [Django v.5.1.3]()
- [Django REST Framework (DRF) v.3.15.2]()
- [Django JWT v.5.3.1]()
- [Celery v.5.4.0]()
- [Django celery-beat v.2.7.0]()
- [Redis v.5.2.0]()
- [CORS v.4.6.0]()

## Использование
#### Для запуска проекта создайте в корне проекта файл .env следующего вида:
```commandline
# настройки Джанго
SECRET_KEY=***

# настройки БД
NAME=***
USER_DB=postgres
PASSWORD=***
HOST=127.0.0.1
PORT=5432

# настройки редис для Celery
CELERY_BROKER_URL=***
CELERY_RESULT_BACKEND=***

# настройки телеграмм-бота
BOT_TOKEN=***
```
